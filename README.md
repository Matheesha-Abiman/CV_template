## ğŸš€ Getting Started

You can use this template either online (easiest) or locally on your machine.

### Option A: Use with Overleaf (Recommended)
1. Go to [Overleaf.com](https://www.overleaf.com) and create a free account.
2. Click **New Project** > **Blank Project**.
3. Copy the code from `resume.tex` in this repository.
4. Paste it into the Overleaf editor, replacing everything.
5. Click **Recompile** and download your PDF!

### Option B: Use with VS Code
1. Install a LaTeX distribution like **MiKTeX** (Windows) or **MacTeX** (Mac).
2. Install the **LaTeX Workshop** extension in VS Code.
3. Open the `.tex` file and press `Ctrl+Alt+B` to build.

## âœ… ATS-Friendly Features
This template is specifically designed to pass through **Applicant Tracking Systems (ATS)**:
* **No Tables or Columns:** Complex layouts often break ATS scanners.
* **Standard Headings:** Uses common keywords that scanners look for.
* **Text-Based:** Since it's compiled from LaTeX, the text is perfectly selectable and readable by machines.
